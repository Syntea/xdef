Provides USER INTERFACE for accessing to XMDefinition objects (models). For In the case you want to know properties of XMdefinition objects of an processed object you can use methods {@link cz.syntea.xdef.proc.XXNode#getXMDefinition()} or {@link cz.syntea.xdef.proc.XXElement#getXMElement()} or {@link cz.syntea.xdef.proc.XXData#getXMData()}.

All methods above return an object with properties of processed data. See:
