# Documentation Directory

This is a directory with complete documentation.

Original source code are doc-files. Pdf-files are generated from doc-files.

Description of individual files follow:
  * "xdef-4.2"
    * X-definition language description
  * "xdef-4.2_construction_mode"
    * introduction to construction mode of X-definition
  * "xdef-4.2_Programming"
    * introduction to X-definition Java programming
  * "xdef-4.2_Xcomponent"
    * introduction to X-definition X-component
  * "xdef-4.2_Lexicon"
    * introduction to X-definition X-lexicon
  * "xdef-4.2_manual.pdf"
    * X-definition user manual
